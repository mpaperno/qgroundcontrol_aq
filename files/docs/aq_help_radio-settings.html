AutoQuad Radio Controls Setup

To check proper RC radio operation, use the RC Telemetry feature when connected to AutoQuad. This will send back the channel values AQ is receiving in real-time.

NOTE that the Pitch channel negative is forward, positive is back

With some AQ firmwares you can customize switch values by selecting the Advanced Switch Cfg. checkbox on this page. This can be useful for setting up more than three controls per channel (eg. to have all flight modes on one radio channel).
